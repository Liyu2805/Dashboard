# SimpleJS - Profile Dashboard

This is an interactive dashboard project that allows users to create a profile and view it on a personalized dashboard. It includes a modal feature to simulate project creation.

## Features
- Profile creation form with fields: Name, Email, Skill Level, and Bio
- Dashboard that displays saved profile information
- Modal popup for creating a project
- Responsive and modern dark-themed design
- Data persistence using browser localStorage

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid for layout)
- JavaScript (for form handling and modal interactions)

## Usage
1. Clone the repository: `git clone <repo-url>`
2. Open `index.html` in a browser.
3. Fill in the form and click 'Continue' to navigate to the dashboard.
4. Click 'Create Project' to view the modal interaction.

## Purpose
This project demonstrates the ability to create interactive web interfaces using HTML, CSS, and JavaScript, with user-friendly forms, data persistence, and modal popups.
